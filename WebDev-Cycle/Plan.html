Planning includes creating a site map and a wireframe that illustrate a connected, dynamic, easy-to-use website.

A sitemap shows the entire hierarchy of a website. It allows people to see how you'd navigate the website and how information inside the website would flow.

A wireframe shows the layout each specific page of a website will have. It includes interactive elements on the pages as well as any places where pictures or graphics will go.
