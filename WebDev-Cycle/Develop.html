The development step is where the code for the website is written. It can use as many programming languages as it wants, often including HTML, CSS, JavaScript, and more. The pages are often created in the order of the heirarchy seen in the sitemap, starting with the home page.

There is both front-end development (what the user is able to see and interact with) and back-end development (everything behind the scenes). It is also where you ensure users can find your website when searching for it online.
